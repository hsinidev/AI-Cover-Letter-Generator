## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have the following installed on your machine:
-   [Node.js](https://nodejs.org/) (v18 or later)
-   `npm`, `pnpm`, or `yarn` package manager

### Installation & Setup

1.  **Get a Gemini API Key**
    You can get one for free from [Google AI Studio](https://aistudio.google.com/).

2.  **Clone the Repository**
    ```sh
    git clone https://github.com/hsinidev/ai-cover-letter-generator.git
    cd ai-cover-letter-generator
    ```

3.  **Install Dependencies**
    ```sh
    npm install
    ```

4.  **Set Up Environment Variables**
    Create a file named `.env` in the root of the project and add your Gemini API key:
    ```env
    # Make sure to name it exactly like this for the service to pick it up
    API_KEY=your_gemini_api_key_here
    ```

5.  **Run the Development Server**
    ```sh
    npm start
    ```
    The application will be available at `http://localhost:3000` (or the next available port).
